백준 알고리즘 (Java)

1002-터렛

1. 문제 설명

  • https://www.acmicpc.net/problem/1002

    2. 접근 방법

문제 파악

  • 한 좌표에서 r만큼 떨어진 점들의 집합 => 원
  • 조규현 기준으로 류재명이 있을 수 있는 좌표의 집합 => 중심이 (x1, y1)이고 반지름의 길이가 r1인 원
  • 백승환 기준으로 류재명이 있을 수 있는 좌표의 집합 => 중심이 (x2, y2)이고 반지름의 길이가 r2인 원

사용할 알고리즘

  • 가능한 모든 경우를 그려본 후 경우에 따라 케이스를 분리한다.
  • 기하학, 수학, 구현

3. 문제 풀이

  • dist는 두 원 사이의 거리로 정의한다. (Math.sqrt(Math.pow(x2 - x1, 2) + Math.pow(y2 - y1, 2)))

  • 두 원이 동일할 때 (원의 중심과 반지름의 길이 모두 같을 때)

    • 두 원일 동일할 경우 류재명이 있을 수 있는 점들의 집합은 두 원과 같다. (무한대의 경우이다)
    • 두 원의 중심과 반지름의 길이가 모두 같은 경우 (x1 == x2 && y1 == y2 && r1 == r2)

  • 두 원이 만나지 않을 때

    • 두 원이 만나지 않을 경우 류재명이 있을 수 있는 점들의 집합은 공집합이다. (있을 수 있는 경우가 존재하지 않는다)

    • 두 원이 만나지 않는 경우는 다음과 같다.

      1. 두 원이 멀리 떨어져 있는 경우 (dist > r1 + r2)
      2. 한 원이 다른 하나의 원 안에 있는 경우 (dist + r1 < r2 || dist + r2 < r1)

  • 두 원이 한 접점에서 만날 때

    • 두 원이 한 접점에서 만날 경우 류재명이 있을 수 있는 점은 하나이다.

    • 두 원이 한 점에서 만나는 경우는 다음과 같다.

      1. 한 원이 다른 원의 외부에서 접하는 경우 (dist == r1 + r2)
      2. 한 원이 다른 원의 내부에서 접하는 경우 (dist + r1 == r2 || dist + r2 == r1)

  • 그 외

    • 그 외에는 두 원이 두 접점에서 만난다.

위의 경우를 코드로 표현하면 아래와 같다.

4. 코드

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.StringTokenizer;

public class Main {
  public static void main(String args[]) throws NumberFormatException, IOException {
    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
    int T = Integer.parseInt(br.readLine());

    for (int i = 0; i < T; i += 1) {
      StringTokenizer line = new StringTokenizer(br.readLine());
      int x1 = Integer.parseInt(line.nextToken());
      int y1 = Integer.parseInt(line.nextToken());
      int r1 = Integer.parseInt(line.nextToken());
      int x2 = Integer.parseInt(line.nextToken());
      int y2 = Integer.parseInt(line.nextToken());
      int r2 = Integer.parseInt(line.nextToken());
      // Distance between (x1, y1) and (x2, y2)
      double dist = Math.sqrt(Math.pow(x2 - x1, 2) + Math.pow(y2 - y1, 2));
      // Case infinite => two circles are same
      if (x1 == x2 && y1 == y2 && r1 == r2) {
        System.out.println(-1);
      }
      // Case 0 => two circles are never meet
      else if (dist > r1 + r2 || (dist + r1 < r2 || dist + r2 < r1)) {
        System.out.println(0);
      }
      // Case 1 => two circles meet only one dot(접하는 경우)
      else if(dist + r1 == r2 || dist + r2 == r1 || dist == r1 + r2) {
        System.out.println(1);
      }
      // Case 2 => Else
      else {
        System.out.println(2);
      }

    }
  }
}
